# 从零搭建个人博客：技术选型与架构设计

## 为什么自建博客

> 自建博客不仅是为了展示，更是为了学习。

## 技术选型

### 前端技术栈

| 技术 | 选择理由 |
|------|----------|
| Vue 3 | 组合式 API 灵活，生态成熟 |
| Vite | 开发体验极佳，构建速度快 |
| Element Plus | 组件丰富，文档完善 |

### 后端技术栈

| 技术 | 选择理由 |
|------|----------|
| Spring Boot 3 | 快速开发，生态完善 |
| MyBatis-Plus | 简化数据库操作 |
| Redis | 缓存热门数据 |
| MySQL | 稳定可靠的关系型数据库 |

## 架构设计

```text
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   Vue 3     │────▶│  Nginx      │────▶│ Spring Boot │
│   前端       │     │  反向代理    │     │   后端       │
└─────────────┘     └─────────────┘     └──────┬──────┘
                                               │
                                      ┌────────┴────────┐
                                      │  MySQL   Redis   │
                                      └─────────────────┘
```

## 功能模块

1. **文章管理**：Markdown 编辑、标签分类、搜索
2. **专题系统**：按主题组织文章
3. **评论互动**：基于 Token 的认证机制
4. **友链**：互换友情链接

## 部署流程

```bash
# 构建前端
cd frontend && npm run build

# 构建后端
cd backend && mvn package

# Docker 部署
docker-compose up -d
```

> 做自己的项目是学习技术最好的方式。
